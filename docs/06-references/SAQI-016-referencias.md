## 15. Criterios de Inclusión

- **Prioridad**: Fuentes primarias (papers revisados por pares, estándares oficiales, documentación autoritativa)
- **Año**: Preferencia 2020+ para temas IA/agentes; clásicos para fundamentos SE
- **Accesibilidad**: DOI, arXiv, URLs oficiales; no referencias rotas
- **Relevancia**: Citadas directamente en documentos SAQI-001 a SAQI-015

- Documentación de versiones obsoletas
- Papers no relacionados con SE, QA, IA, o bases de datos
- Referencias inventadas o no verificables
